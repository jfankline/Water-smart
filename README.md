# AI Water Leak & Rain-Harvest Manager - Backend

## Overview
Backend API for the AI Water Leak & Rain-Harvest Manager application. Provides water management, leak detection, and drought planning functionality for smallholder farmers.

## Features
- User authentication and management
- Water source tracking
- AI-powered leak detection
- Rainfall monitoring
- Water usage analytics
- Drought planning tools
- SMS/WhatsApp alerts
- Admin dashboard for NGOs/governments

## Tech Stack
- **Framework**: Flask (Python)
- **Database**: PostgreSQL / SQLite
- **AI/ML**: scikit-learn (Isolation Forest)
- **Authentication**: JWT tokens
- **API Documentation**: OpenAPI/Swagger (planned)
